NuxtJs的安装

在官网下载nodejs的zip版本；
http://nodejs.cn/download/
解压并在根目录下新建两个文件夹node-cache 和node-global,用于设置缓存路径和全局包路径；
12.13.0版本在运行 npm install 命令时会自动创建；

执行命令
npm config set cache “E:\nodeInstall\node-v10.14.0-win-x64\node-cache”
npm config set prefix “E:\nodeInstall\node-v10.14.0-win-x64\node-global”

解压版目录：
node_modules: npm 默认安装时候组件都会放在这里，在每一个组件的目录下会存在一个
package.json文件，npm install 命令会根据这个来进行更新；

修改配置环境变量（创建当前用户的环境变量会最小程度影响系统，但是其他用户不会生效）
在桌面右键我的电脑（或计算机）—属性 —高级系统设置—-环境变量—-
添加NODE_HOME :E:\nodeInstall\node-v10.14.0-win-x64 (你的nodejs安装根目录，如下图)
path添加: ;%NODE_HOME%;%NODE_HOME%\node-globa;

安装淘宝源的cnpm
npm install -g cnpm –registry=https://registry.npm.taobao.org

更换成淘宝的源
npm config set registry https://registry.npm.taobao.org
– 配置后可通过下面方式来验证是否成功
npm config get registry
– 或npm info express

nrm切换源

cnpm不能作为psl命令执行的处理方式
set-ExecutionPolicy RemoteSigned

取消ssl证书认证
npm config set strict-ssl false//取消npm的https认证

版本12.13.0已经带有npx

npx create-nuxt-app first-app

nuxtjs插件配置
安装
npm install @nuxtjs/axios

配置nuxt.config.js
export default{
modules: [
// Doc: https://bootstrap-vue.js.org
‘bootstrap-vue/nuxt’,
‘@nuxtjs/axios’,
]
/ 需要使用 aixos必须配置以下两项axios和proxy/
axios: {
prefix: ‘/api/‘,
proxy: true
},

proxy: {
    '/api/': {
        target: 'https://maoyan.com/',
        pathRewrite: {
        '^/api/': ''
        }
    }
}

}
