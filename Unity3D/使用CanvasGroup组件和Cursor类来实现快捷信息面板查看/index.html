使用CanvasGroup组件和Cursor类来实现快捷信息面板查看

主要用到CanvasGroup组件和Cursor类

CanvasGroup是Canvas中的一个可选组件,添加Canvas时候自动附加:CanvasGroup
通过Interactable 属性来设置其激活状态,true时接受鼠标和键盘的操作, false时不接受鼠标和键盘的操作
通过alpha属性设置透明度,1为完全透明,0为完全不透明

Cursor类定义以及实例:Scripting API

有关CursorLockMode:
None: 鼠标移动方式不受限制,相当于自由移动,包括游戏窗口之外
Locked: 固定在游戏窗口之中,且为正中位置
Confined: 限制在游戏窗口之中,大多数PC正常设置应该为此项

使用一下代码在TAB键松开时候自动隐藏并且取消面板激活状态
