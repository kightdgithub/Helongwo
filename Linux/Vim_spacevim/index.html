为VIM 安装 spacevim 插件

为VIM 安装 spacevim 插件

需要安装vimproc
https://github.com/Shougo/vimproc.vim#building
将vimproc下载之后按照步骤解压编译，然后修改文件名称为vimfiles, 存放位置为Home目录下，一般就是通过在vim中使用vimruntimepath查询出来的路径

vim 修改时提示 E21: Cannot make changes, ‘Modifiable’ is off

给不能连接外网的机器安装spacevim： https://blog.csdn.net/u013033845/article/details/90904592

https://spacevim.org/cn/quick-start-guide/#%E5%AE%89%E8%A3%85%E6%8C%87%E5%8D%97

curl -sLf https://spacevim.org/cn/install.sh | bash

SpaceVim中vimproc的vimproc_linux64.so未找到：https://www.cnblogs.com/zhuxiaoxi/p/8466846.html

安装 nerd-font: https://github.com/ryanoasis/nerd-fonts#option-3-install-script

字体的安装，修改spacevim的配置文件，将guifont设置为’DejaVu Sans Mono for Powerline’，将linux中的字体拷贝到windows下的字体目录内windows/fonts，putty设置使用此字体

使用console-setup重新配置
https://www.cnblogs.com/lzyws739307453/p/11795048.html
